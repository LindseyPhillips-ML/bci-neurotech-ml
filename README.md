# bci-neurotech-ml

Machine learning projects focused on neural signal decoding and brain-computer interface applications, such as EEG decoding, spike sorting, and real-time BCI pipelines. Built as part of a self-directed portfolio targeting MLE roles in neurotech and BCI.

---

## Focus Areas

- EEG motor-imagery decoding
- Cross-subject transfer learning for neural signals
- Neural data artifact removal and preprocessing
- Spike sorting on implant-style neural recordings
- Real-time signal-to-action pipelines
- Lossless neural data compression

## Tech Stack

Python · PyTorch · MNE · NumPy · SciPy · scikit-learn
